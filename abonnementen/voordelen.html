Waarom MyFamTreeCollab?

Een moderne, privacy-first stamboomtool die groeit met jouw behoeften. Van lokaal beheer tot cloud-samenwerking — jij bepaalt wanneer en hoe.

Onze voordelen

🔐

Volledige controle over je data

Jouw stamboom blijft standaard lokaal in je browser. Geen verplicht account, geen tracking.

  • ✓ Data lokaal opgeslagen
  • ✓ Geen verplicht account
  • ✓ Jij bepaalt wat gedeeld wordt
📤

Altijd exporteerbaar

Je data zit nooit vast in het systeem. Exporteren kan altijd in open formaten.

  • ✓ CSV export
  • ✓ JSON export
  • ✓ GEDCOM voorbereid (toekomst)
🌳

Heldere stamboomweergave

Bouw en bekijk je familiegeschiedenis op een visuele manier.

  • ✓ Stamboom view
  • ✓ Tijdlijn weergave
  • ✓ Statistieken dashboard
👥

Flexibel rolmodel

Start gratis lokaal en groei naar cloud-samenwerking met het rolmodel dat bij jou past.

  • ✓ Gratis lokaal gebruik
  • ✓ Viewer · Editor · Owner rollen
  • ✓ Uitnodigen via e-mail
🤝

Samen bouwen

Werk samen met familieleden aan één gedeelde stamboom via berichten en wijzigingen.

  • ✓ Gedeelde stambomen
  • ✓ Berichtenboard per persoon
  • ✓ Versiebeheer per persoon
