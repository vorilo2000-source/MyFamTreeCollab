Toegang & Samenwerken

MyFamTreeCollab werkt met account types die bepalen wat je kan doen.

⚙️ Hoe werkt toegang?

1
Niet ingelogd starten — iedereen kan direct een demo-stamboom bekijken zonder account.
2
Basis account (guest) aanmaken — gratis registreren geeft toegang tot max. 60 personen lokaal.
3
Owner account — Basis account + donatie, activeert cloud backup, delen en samenwerken.
4
Viewer / Editor toewijzen — een Owner nodigt anderen uit en geeft ze lees- of bewerkrecht op een specifieke stamboom.

Account types

👤 Niet ingelogd
tier: null
Geen account vereist

Bezoeker zonder account. Kan alleen een demo-stamboom bekijken. Geen opslag, geen functies.

  • ✓ Demo-stamboom bekijken
  • ✗ Geen eigen stamboom
  • ✗ Geen opslag
  • ✗ Geen functies
🖥️ Basis account
tier: guest
Gratis — geen donatie vereist

Gratis account. Lokale stamboom tot max. 60 personen. Geen cloud, geen samenwerking.

  • ✓ Lokale stamboom (max. 60 personen)
  • ✓ Tijdlijn & statistieken
  • ✓ Export naar CSV / JSON
  • ✗ Geen cloud backup
  • ✗ Geen stamboom delen
  • ✗ Geen viewer/editor uitnodigen
👑 Owner account
tier: owner
Donatie vereist

Volledige toegang. Cloud backup, onbeperkte personen, stamboom delen en samenwerking activeren.

  • ✓ Onbeperkt personen lokaal
  • ✓ Cloud backup
  • ✓ Stamboom delen
  • ✓ Viewer / Editor uitnodigen
  • ✓ Versiebeheer (max. 5 versies)
  • ✓ Samenwerken (collab)

📋 Stamboom-rechten: Viewer & Editor

👁️ Viewer

Leesrecht op een gedeelde stamboom. Toegewezen door de Owner van die stamboom.

  • ✓ Stamboom bekijken
  • ✓ Tijdlijn & statistieken
  • ✓ Berichten in discussies
  • ✗ Geen bewerkingen
✏️ Editor

Bewerkrecht op een gedeelde stamboom. Toegewezen door de Owner van die stamboom.

  • ✓ Alles van Viewer
  • ✓ Personen toevoegen & bewerken
  • ✓ Relaties beheren
  • ✓ Wijzigingsvoorstel plaatsen
