Waarom MyFamTreeCollab?

Een moderne, privacy-first stamboomtool die groeit met jouw behoeften.

Onze voordelen

🔐

Volledige controle over je data

Jouw stamboom blijft standaard lokaal in je browser.

  • ✓ Data lokaal opgeslagen
  • ✓ Geen verplicht account
  • ✓ Jij bepaalt wat gedeeld wordt
📤

Altijd exporteerbaar

Je data zit nooit vast in het systeem.

  • ✓ CSV export
  • ✓ JSON export
  • ✓ GEDCOM voorbereid (toekomst)
🌳

Heldere stamboomweergave

Bouw en bekijk je familiegeschiedenis op een visuele manier.

  • ✓ Stamboom view
  • ✓ Tijdlijn weergave
  • ✓ Statistieken dashboard
👤

Flexibel account model

Start gratis als basis account (guest) en groei naar een Owner account via een kleine donatie.

  • ✓ Basis account — gratis, lokaal, 60 personen
  • ✓ Owner account — cloud, onbeperkt, samenwerken
  • ✓ Uitnodigen via e-mail als viewer/editor
🤝

Samen bouwen

Werk samen met familieleden aan één gedeelde stamboom.

  • ✓ Gedeelde stambomen (Owner)
  • ✓ Berichtenboard per persoon
  • ✓ Versiebeheer per persoon
