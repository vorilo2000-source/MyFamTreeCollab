Waarom MyFamTreeCollab?

Een moderne, privacy-first stamboomtool die groeit met jouw behoeften. Van gratis lokaal beheer (basis account) tot cloud-samenwerking (Owner account) — jij bepaalt wanneer en hoe.

Onze voordelen

🔐

Volledige controle over je data

Jouw stamboom blijft standaard lokaal in je browser. Geen verplicht account, geen tracking, geen cloud tenzij jij dat wilt.

  • ✓ Data lokaal opgeslagen
  • ✓ Geen verplicht account
  • ✓ Jij bepaalt wat gedeeld wordt
📤

Altijd exporteerbaar

Je data zit nooit vast in het systeem. Exporteren kan altijd in open formaten, ook met een gratis basis account.

  • ✓ CSV export
  • ✓ JSON export
  • ✓ GEDCOM voorbereid (toekomst)
🌳

Heldere stamboomweergave

Bouw en bekijk je familiegeschiedenis op een visuele manier. Beschikbaar voor elk account type.

  • ✓ Stamboom view
  • ✓ Tijdlijn weergave
  • ✓ Statistieken dashboard
👤

Flexibel account model

Start gratis als basis account (guest) en groei naar een Owner account via een kleine donatie (~€5 per 6 maanden) wanneer je wilt samenwerken.

  • ✓ Basis account — gratis, lokaal, 60 personen
  • ✓ Owner account — cloud, onbeperkt, samenwerken
  • ✓ Uitnodigen via e-mail als viewer/editor
🤝

Samen bouwen

Werk samen met familieleden aan één gedeelde stamboom. Alleen beschikbaar voor Owner accounts — jij bepaalt wie toegang heeft.

  • ✓ Gedeelde stambomen (Owner)
  • ✓ Berichtenboard per persoon
  • ✓ Versiebeheer per persoon
